# 周末规划器 (Weekend Todo List) - 设计文档

**创建日期：** 2025-03-15

## 概述

一个帮助打工人（尤其是996）提前规划周末/假期的轻量级应用。通过预设目标、奖励系统和多巴胺反馈，提升完成率和幸福感，避免时间虚度带来的懊悔感。

## 核心定位

- **目标用户：** 打工人，尤其是996人群，自控力较弱需要外部约束
- **核心价值：** 提前规划 → 完成任务 → 获得奖励 → 感受满足
- **使用场景：** 周末/假期规划，工作日计划为次要功能

## 技术栈

- **前端：** Vue 3 + Vite
- **数据存储：** localStorage（当前数据）+ IndexedDB（历史记录）
- **移动端：** PWA 支持，可离线使用
- **未来扩展：** Node.js 后端接口层已预留

## 数据结构

```typescript
// 周末计划
WeekendPlan {
  id: string
  startDate: Date
  endDate: Date
  days: DayPlan[]
  totalPoints: number
  completedPoints: number
}

// 每日计划
DayPlan {
  date: Date
  tasks: Task[]
  dailyGoal: number      // 每日目标完成率 %
  reward?: Reward
  completedPoints: number
}

// 任务
Task {
  id: string
  title: string
  category: string       // 运动、休闲、家务、美食等
  priority: 'high' | 'medium' | 'low'
  note?: string
  points: number
  completed: boolean     // 只关注是否完成，无时间预估
}

// 奖励
Reward {
  id: string
  title: string
  pointsRequired: number
  redeemed: boolean
}
```

## 功能模块

### 1. 计划设置模块
- 模板系统：预设模板（休闲型、充实型、运动型、平衡型）+ 自定义模板
- 日期选择：选择周末/假期日期范围
- 快速填充：一键复制上周计划
- 手动添加：逐个添加任务（标题、分类、优先级、备注、积分）

### 2. 任务管理模块
- 任务列表：按日期展示，分类和优先级视觉区分
- 手势操作：左滑完成、右滑删除
- 批量操作：多选完成/删除
- 任务编辑：修改任何属性

### 3. 奖励系统模块
- 积分获取：完成任务立即获得积分 + 数字跳动动画
- 每日目标：设置每日完成率目标，达成解锁奖励
- 奖励池：预设奖励列表（奶茶、游戏时间、电影等），积分兑换
- 兑换记录：查看历史兑换

### 4. 反馈与展示模块
- 完成动画：烟花/粒子效果
- 等级系统：积分累计升级，显示等级徽章
- 每日总结卡片：当天完成度、获得积分、解锁奖励
- 每周总结：完成率趋势、成就统计

### 5. 数据管理模块
- 导出：JSON 格式（数据迁移）+ Markdown 格式（人类可读）
- 导入：从文件恢复数据
- 历史记录：IndexedDB 存储所有历史计划
- 数据统计：完成率趋势、最常做任务、积分趋势图

## 页面结构

### 1. 首页 - 当前周末
- 顶部：日期范围、总进度条、总积分
- 中间：按 Tab 切换每天的任务列表
- 底部：添加任务浮动按钮
- 右上角：菜单入口

### 2. 计划设置页
- 顶部：日期选择器
- 模板选择区：卡片式展示预设模板
- 任务编辑区：列表展示每天任务
- 底部：保存/取消按钮

### 3. 奖励页
- 顶部：当前积分、等级、进度条
- 每日目标奖励区
- 奖励池：可兑换奖励列表
- 兑换记录区

### 4. 历史与统计页
- Tab 切换：历史记录 / 数据统计
- 历史记录：时间轴展示
- 数据统计：完成率趋势图、积分趋势图、最常做任务 Top 5

### 5. 设置页
- 数据导出/导入
- 清空数据
- 主题切换（预留）
- 关于

### 底部导航
- 📅 本周（首页）
- ✅ 计划
- 🎁 奖励
- 📊 统计

## 视觉设计

### 配色方案（鲜活渐变）
- 主色渐变：`linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- 背景渐变：`linear-gradient(to bottom, #f5f7fa 0%, #c3cfe2 100%)`
- 卡片：纯白 + `box-shadow: 0 4px 20px rgba(0,0,0,0.08)`

### 分类标签（渐变）
- 运动：`linear-gradient(135deg, #11998e 0%, #38ef7d 100%)`
- 休闲：`linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- 家务：`linear-gradient(135deg, #f093fb 0%, #f5576c 100%)`
- 美食：`linear-gradient(135deg, #fa709a 0%, #fee140 100%)`

### 优先级
- 高优先级：`#FF6B6B`
- 中优先级：`#FFA500`
- 低优先级：`#4ECDC4`

### 交互动画
- 完成任务：缩放 + 淡出 + ✅ 勾选（300ms）
- 获得积分：数字滚动（500ms）
- 等级提升：全屏烟花（2s）
- 页面切换：淡入淡出（200ms）

## 数据持久化

### 存储策略
- `localStorage`：当前编辑的周末计划
- `IndexedDB`：历史记录 + 统计数据
- 定期清理：保留最近 3 个月，更早的归档
