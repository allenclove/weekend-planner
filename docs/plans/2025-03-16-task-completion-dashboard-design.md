# 任务完成交互与看板设计

## 概述
增强任务完成的交互体验，添加完成特效，并提供完善的成就看板。

## 功能一：点击整行完成

### 交互改进
- 整个任务行变为可点击区域
- 点击任意位置（除删除按钮外）切换完成状态
- 保留复选框作为视觉指示器

### 实现方式
```vue
<div class="task-item cursor-pointer" @click="toggleTask">
  <input type="checkbox" ... @click.stop />
  <div>任务内容</div>
  <button @click.stop>删除</button>
</div>
```

## 功能二：粒子爆炸特效

### 实现方案
- 12个粒子从任务中心向外爆炸
- 每个粒子随机角度、颜色（金/白/灰）
- 动画时长 600ms
- 同时任务文字产生划线动画

### 技术细节
- CSS `@keyframes` 定义爆炸轨迹
- CSS 变量随机化粒子角度和距离
- `pointer-events: none` 不干扰交互

## 功能三：混合视图看板

### 页面结构
```
┌─────────────────────────────┐
│  📊 成就看板                │
├─────────────────────────────┤
│  统计卡片：总完成/本周/连续  │
│  最常完成：任务频率排行      │
│  历史记录：卡片式列表        │
└─────────────────────────────┘
```

### 数据来源
- 总完成数：遍历所有历史记录
- 本周数：过滤本周计划
- 连续天数：计算相邻有记录日期
- 最常完成：按任务标题分组统计
